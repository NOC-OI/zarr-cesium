ColorMapName

type ColorMapName = typeof allColorScales[number];

Type representing valid color map names. The values are derived from the allColorScales array imported from the jsColormaps module and are based on matplotlib colormap (https://matplotlib.org/stable/users/explain/colors/colormaps.html).

The jsColormaps module was adapted from the jsColormaps project (https://github.com/timothygebhard/js-colormaps) which provides JavaScript implementations of various colormaps.
